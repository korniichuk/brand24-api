## Portability
brand24-api checked on Ubuntu Desktop 16.04 LTS.

## Requirements
Please, install `chromium-chromedriver` package:
```
$ sudo apt-get install chromium-chromedriver
```

Next, install Python packages:
```
$ sudo pip install -r requirements.txt
```
